# 📊 Sales Dashboard App

This is a live sales analytics dashboard built with HTML, JavaScript, and Google Apps Script.

---

## 🚀 Features

- 🔐 Simple login system (shared password)
- 📊 Top 5 products for selected date range
- 📈 Daily sales line chart
- 📦 Full product breakdown table
- ⚡ Fast API-based data loading
- 📱 Mobile-friendly UI

---

## 🔑 Login

Default demo passwords:

- 1234

- staff2

---

## 📡 Data Source

This app connects to a Google Apps Script API which provides sales data in JSON format.

---

## 🌍 Hosting

This app is designed to run on GitHub Pages (free hosting).

## ⚠️ Notes

- This is an internal dashboard tool
- Not designed for public access security
- Best used for small teams or personal business tracking
